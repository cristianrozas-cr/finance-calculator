1. Crear el repo local (Directorio / file py)
2. Inicializar el repo local = git init
3. Crear el repositorio en la nube (en github)
4. Usar los comandos en la terminal de vs code para subir el repositorio
    1. git init
    2. git add .
    3. git commit -m “first commit”
5. Sincronizar con el repo en github = git remote add origin https://……
6. Crear el ambiente virtual (enviroment), escribir en la terminal lo siguiente:
    
    ```bash
    python -m venv env //para crear el ambiente
    
    /scripts/activate //dentro del directorio de ambiente, escribir este código
    ```
    
7. Ejecutar el comando git push -u origin main
